## Description
OrganicGrow is a web-based platform to educate farmers about organic farming,
including crop-specific tips, natural fertilizers, and pest control methods.

## Features
- Crop-wise organic farming tips
- Natural fertilizers guide
- Organic pest control techniques
- Simple navigation with frontend and backend integration

## Technologies Used
- Frontend: HTML, CSS
- Backend: Java Spring Boot
- Database: Optional MySQL/PostgreSQL

## Future Enhancements
- Farmer login & profile
- Admin panel to update content
- Search functionality for crops and fertilizers
- Mobile app integration
